# Mahmoud's Portfolio Website

A modern professional portfolio built with React, Tailwind CSS, and Framer Motion.

## 🚀 Features
- Responsive design
- Dark/Light mode
- Smooth animations
- SEO optimized
- Glassmorphism effects
- RTL Arabic support
- Contact form + WhatsApp integration
- Resume download

## 🛠 Tech Stack
- React
- Tailwind CSS
- Framer Motion
- React Icons

```bash
git clone https://github.com/yourusername/portfolio-website.git
